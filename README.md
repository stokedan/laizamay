# Girlfriend Gift Picker (Tops/Bottoms/Socks)

A small mobile-friendly app where she picks:

- **Category**: tops / bottoms / socks
- **Brand**
- **Item** (your curated options)

Then the app shows a **reveal** screen.

## Run it locally

```bash
cd girlfriend-gift-app
npm install
npm run dev
```

Open the local URL shown in the terminal (usually `http://127.0.0.1:5173/`).

## Build a shareable version

```bash
npm run build
```

The production build outputs to `dist/`.
